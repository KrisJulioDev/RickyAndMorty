# RickyAndMorty

Sample app that display rick and morty characters 
Created in SwiftUI, MVVM architecture
Features includes:
- Loading of characters
- Display of detailed view of character
- Infinite pagination
- Offline support
- Mark as favourite (not completed due to time constraints)
